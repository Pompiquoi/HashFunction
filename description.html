Detailed Description

Hash functions transform input data into fixed-length outputs. Common threats include collision attacks, where two inputs produce the same hash, and pre-image attacks, targeting reverse engineering of the input.

Example with Hashcat

# Cracking MD5 Hash with Hashcat
hashcat -a 0 -m 0 hash.txt wordlist.txt

This command uses Hashcat to attempt cracking an MD5 hash using a wordlist.
